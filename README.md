
# 🛍️ 패션 플랫폼 데이터 분석 (Fashion-Platform-Trend-Analysis)

무신사(Musinsa) 데이터를 기반으로 최신 패션 트렌드와 소비자의 구매 결정 요인을 분석하고, 실질적인 비즈니스 전략을 제언하는 프로젝트입니다.

## 🎯 프로젝트 목표

> 무신사 랭킹 데이터를 통해 **무엇이** 유행하는지 파악하고, 가격, 세일, 사용자 리뷰 등을 깊이 파고들어 **왜** 유행하는지를 데이터 기반으로 설명하며
>  이를 바탕으로 무신사와 브랜드들의 실질적인 비즈니스 전략을 제언하는 것입니다.

---

## 📥 1. 사용 데이터 (Data to be Used)

- **데이터 소스**: `무신사(musinsa.com)` 웹사이트
- **수집 방법**: `Python`을 이용한 웹 크롤링(Web Crawling)
- **수집 규모**:
    - **7개** 카테고리: 상의, 아우터, 바지, 신발, 가방, 패션소품, 뷰티
    - **2개** 랭킹 종류: 월간 / 주간
    - 상품 **상위 30개**
    - 리뷰 **상품당 30**
    - **결과**: 약 **420개** 상품 정보 및 약 **12,600**의 리뷰 데이터
- **수집 필드**:
    - 브랜드명, 상품명, 정상가, 할인가, 할인율, 리뷰 텍스트, 사용자 신체 정보(키/몸무게), 구매 사이즈 등

---

## 🛠️ 2. 사용 기술 (Technology to be Used)

- **데이터 수집**: `Python`, `BeautifulSoup`, `Selenium`
- **데이터 처리 및 분석**: `Pandas`, `Konlpy` (한글 자연어 처리)
- **데이터 시각화**: `Matplotlib`, `Seaborn`, `WordCloud`
- **프로젝트 관리 및 보고**: `Jupyter Notebook`, `GitHub`

---

## 📈 3. 프로젝트 범위 (Project Scope)

### 분석 관점
- 과거 데이터 추적이 아닌, **현재 시점의 스냅샷(Snapshot) 분석**에 집중합니다.
- 월간 랭킹과 주간 랭킹을 비교하여 시장의 동향을 파악합니다.

### 핵심 분석 과제
1. 랭킹 비교를 통한 **스테디셀러와 급상승 아이템 식별**
2. **가격 및 할인율**이 랭킹 순위에 미치는 영향 분석
3. 리뷰의 **핵심 키워드 분석**을 통한 구체적인 인기 요인 도출
4. 분석 결과를 바탕으로 한 **비즈니스 전략 제언**

### 최종 결과물
- 분석 과정, 시각화 자료, 비즈니스 제언을 포함한 **GitHub 기반의 포트폴리오**

## 🤖 4. 생성형 AI 결과 (Gemini의 Deep Research 사용)

### 생성형 AI  질문한 내용
    당신은 패션 이커머스 트렌드를 전문적으로 분석하는 데이터 분석가입니다.

    2025년 9월 현재, '무신사' 플랫폼의 데이터를 분석하는 프로젝트를 막 완료했다고 가정해 주세요. 분석 데이터는 7개 주요 카테고리(상의, 아우터, 바지, 신발, 가방, 패션소품, 뷰티)의 '월간' 및 '주간' Top 30 랭킹과, 각 상품별 최신 리뷰 30~50건을 모두 포함합니다.

    이 분석 결과를 바탕으로, 무신사 마케팅 팀에 제출할 '핵심 트렌드 분석 보고서'를 작성해 주세요.

    보고서에는 다음 내용이 반드시 포함되어야 합니다:

    1. **시장 동향 요약**:
    * '스테디셀러'(월간 랭킹)와 '급상승 아이템'(주간 랭킹)의 특징을 비교 분석.

    2. **소비자 인사이트**:
    * 사용자 리뷰에서 도출한 핵심 구매 결정 요인 (가격, 할인율, 핏, 가성비 등).

    3. **비즈니스 제언**:
    * 분석 결과에 기반한 '무신사 플랫폼'과 '입점 브랜드'를 위한 구체적인 실행 전략 제안.
