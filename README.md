
# 🛍️ 패션 플랫폼 데이터 분석 (Fashion-Platform-Trend-Analysis)

무신사(Musinsa) 데이터를 기반으로 최신 패션 트렌드와 소비자의 구매 결정 요인을 분석하고, 실질적인 비즈니스 전략을 제언하는 프로젝트입니다.

## 🎯 프로젝트 목표

> 무신사 랭킹 데이터를 통해 **무엇이** 유행하는지 파악하고, 가격, 세일, 사용자 리뷰 등을 깊이 파고들어 **왜** 유행하는지를 데이터 기반으로 설명하며
>  이를 바탕으로 무신사와 브랜드들의 실질적인 비즈니스 전략을 제언하는 것입니다.

---

## 📥 1. 사용 데이터 (Data to be Used)

- **데이터 소스**: `무신사(musinsa.com)` 웹사이트
- **수집 방법**: `Python`을 이용한 웹 크롤링(Web Crawling)
- **수집 규모**:
    - **7개** 카테고리: 상의, 아우터, 바지, 신발, 가방, 패션소품, 뷰티
    - **2개** 랭킹 종류: 월간 / 주간
    - 상품 **상위 30개**
    - 리뷰 **상품당 30**
    - **결과**: 약 **420개** 상품 정보 및 약 **12,600**의 리뷰 데이터
- **수집 필드**:
    - 브랜드명, 상품명, 정상가, 할인가, 할인율, 리뷰 텍스트, 사용자 신체 정보(키/몸무게), 구매 사이즈 등

---

## 🛠️ 2. 사용 기술 (Technology to be Used)

- **데이터 수집**: `Python`, `BeautifulSoup`, `Selenium`
- **데이터 처리 및 분석**: `Pandas`, `Konlpy` (한글 자연어 처리)
- **데이터 시각화**: `Matplotlib`, `Seaborn`, `WordCloud`
- **프로젝트 관리 및 보고**: `Jupyter Notebook`, `GitHub`

---

## 📈 3. 프로젝트 범위 (Project Scope)

### **Phase 1: 데이터 수집 및 가공 (Data Collection & Processing)**

-   **데이터 수집:** `Python` (`BeautifulSoup`, `Selenium`)을 활용하여 `무신사` 웹사이트의 7개 카테고리에서 월간/주간 Top 30 상품 정보와 상품별 최신 리뷰 30건을 크롤링합니다.
-   **데이터 가공:** `Pandas`를 사용하여 수집된 원시 데이터를 분석 가능한 형태로 정제하고, 숫자 및 텍스트 데이터를 표준화합니다.

### **Phase 2: 탐색적 데이터 분석 (Exploratory Data Analysis)**

-   **시장 동향 분석 (스테디셀러 vs. 급상승 아이템):** 월간 랭킹(안정적 수요)과 주간 랭킹(단기 트렌드)을 비교하여 시장의 장기적 수요와 단기 트렌드를 동시에 파악합니다.
-   **가격 및 할인율 분석:** 랭킹 상위권 상품들의 가격대 및 할인율 분포를 시각화하여 가격 전략을 분석합니다.

### **Phase 3: 심층 분석 및 인사이트 도출 (In-depth Analysis & Insight)**

-   **소비자 구매 요인 분석:** `Konlpy` (한글 자연어 처리)를 활용하여 약 12,600여 개의 리뷰 데이터에서 '핏', '소재', '가성비' 등 소비자의 실질적인 구매 결정 요인을 정량적으로 도출합니다.

### **Phase 4: 결과 종합 및 전략 제언 (Synthesis & Recommendation)**

-   **비즈니스 전략 제언:** 모든 분석 결과를 종합하여 '무신사 플랫폼'과 '입점 브랜드'를 위한 구체적인 마케팅 및 상품 기획 전략을 제언합니다.
-   **최종 결과물:** 전체 분석 과정, 코드, 시각화 자료, 최종 제언을 체계적으로 정리한 **GitHub 기반의 포트폴리오**를 구축합니다.
