# 🛍️ 패션 플랫폼 무신사(Musinsa) 데이터 분석 (Fashion-Platform-Trend-Analysis)

무신사(Musinsa) 데이터를 바탕으로 최신 패션 트렌드와 소비자의 구매 결정 요인을 심층적으로 분석하는 프로젝트입니다.

## 🎯 프로젝트 목표

단순한 상품 순위를 넘어 소비자의 실제 경험이 담긴 리뷰 데이터를 분석하여 트렌드의 본질을 파악하는 것을 목표로 합니다.

> 현재 유행하는 것은 무엇이며고 유행의 이유는 무엇인가?

## 📈 분석 내용

### 1. 랭킹 비교 분석

- `월간 랭킹(Top 30)`을 분석해 시장을 주도하는 안정적인 인기 상품군을 파악합니다.
- `주간 랭킹(Top 30)`을 월간 랭킹과 대조하여 최근 급부상하는 트렌드 아이템의 특징을 분석합니다.

### 2. 리뷰 심층 분석

- `핵심 키워드 분석`: 인기 상품 리뷰에서 가장 빈번하게 언급되는 키워드(핏 소재 가성비 등)를 추출하여 구매 결정 요인을 세밀하게 분석합니다.
- `사이즈 선택 패턴 분석`: 리뷰의 사용자 신체 정보(키/몸무게)와 선택 사이즈 간 상관관계를 분석하여 실질적인 소비자 인사이트를 도출합니다.

## 📥 데이터 수집 계획

- 데이터 소스: `무신사(musinsa.com)`
- 수집 범위:
    - `7개 카테고리`: 상의 아우터 바지 신발 가방 패션소품 뷰티
    - `2개 랭킹`: 월간/주간
    - `상위 30개` 상품
- 수집 규모:
    - `약 420개`의 상품 URL 및 정보
    - `최대 12600건`의 사용자 리뷰 (상품당 최신 30건)
    - 중요한 급상승 아이템만 `50건` 모아 깊게 분석
